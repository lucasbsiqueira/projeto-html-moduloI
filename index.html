Hisória dos Computadores

No ano de 1944 surgi o primeiro computador utilizado para cálculos complexos, com o nome MARK 1 e logo após em 1946 temos o COLOSSUS na Alemanha.
Nessa mesma época surgi o termo tão utilizado na programação "BUG", que se deu devido problemas com insetos que pousavam nos enormes computadores da época e causavam problemas, necessitando de manutenção.
O tamanho das máquinas diminuiram apenas após ocorrer a troca de válvulas por transistores, que mais tarde também foram substituídos, dessa vez por circuitos integrados.
A quarta geração de computadores veio em 1971 com computadores cada vez menores e de uso pessoal.

Clients

Client seria o dispositivo que está consumindo a informação na internet, como notebooks, celulares e etc. Essa informação é enviada através de servidores.
Quando o client realiza o acesso em um site as informações são enviadas como HTML, CSS, JavaScript e arquivos estáticos que seriam imagens, áudios, entre outros.
A velocidade de conexão do client afeta diretamente a experiência do usuário, por conta do tempo que as informações levam para serem exibidas, então um HTML pesado somado a uma velocidade de conexão ruim resultará em uma péssima experiência para o client.
Utilizado com a finalidade de melhorar essa experiência os navegadores implementam o chamado "cache" na aplicação.
Uma vez que uma página é acessada pelo client as informações são armazenadas, para que no próximo acesso essas informação sejam exibidas de maneira mais rápida, evitando um tráfego inútil de informações.

Servers

Servers seriam os servidores que podem ser utilizados tanto na internet como em rede local, como por exemplo em empresas onde é utilizado um computador central para armazenamento de arquivos.
O computador utilizado para ser o servidor deve ser sempre mais robusto, porque demandam grande processamento. Exstem servidores de vários tipos e podemos citar alguns como exemplo:

  • Arquivos;
  • Segurança (Firewall);
  • Streaming;
  • E-mail;
  • WEB;

Como é a Estrutura HTML

HTML na verdade não é uma linguagem de programação! HTML é a sigla de HyperText Markup Language e é basicamente um arquivo de marcação, com textos delimitados por nomes que são interpretados pelo navegador. Cada delimitador possui funções específicas diferentes e chamamos esses delimitadores de tags. A estrututa se inicia no "index" que seria o padrão de início em HTML. Toda tag possui um sinal de < (menor), um sinal de > (maior) e dentro dela uma palavra. O HTML consiste basicamente em duas partes, o corpo (<body>) que são as informações que aparexem para o usuário e o cabeçalho (<head>) que são informações que irão carregar antes de aparecer para o usuário.
